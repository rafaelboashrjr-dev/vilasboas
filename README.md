# Dashboard HRJR - Incidentes de TI

Dashboard web para análise de incidentes de TI do **HRJR - Hospital Regional Jorge Rossmann**.

## Versão

v0.6 - Correção dos filtros e comparação entre anos.

## Recursos

- Upload de Excel/CSV.
- Filtros funcionais.
- Lista completa de meses detectados na base.
- Comparação de incidentes entre anos diferentes.
- KPIs automáticos.
- Gráficos interativos.
- Tabela filtrável.

## Como executar

Abra o arquivo `index.html` no navegador.
