# Dashboard HRJR v2

Versão 2.0 criada em paralelo à versão atual, sem alterar a v1.

## Objetivo

Criar uma interface mais corporativa, próxima de Power BI, com foco em apresentação executiva e gestão à vista.

## Como testar

Abra:

`v2/index.html`

ou pelo GitHub Pages:

`https://rafaelboashrjr-dev.github.io/vilasboas/v2/`

## Entregue nesta primeira fase

- Novo layout executivo.
- Visual mais limpo e corporativo.
- KPIs premium.
- Área hero com status operacional.
- Upload de planilha.
- Cache local da última planilha.
- Gráficos principais.
- Top 10.
- Modo TV.
- PDF executivo inicial.

A v1 permanece preservada na raiz do projeto.
